# Модели картинок НейроБокс — функционал и возможности

## Провайдеры: прямые API и fal.ai

Маршрутизация по приоритету:

| Приоритет | Условие | Провайдер |
|-----------|---------|------------|
| 1 | Модель DALL-E 3 или GPT Image | **Прямой OpenAI API** (`openai_api_key`) |
| 2 | Модель Nano Banana (и pro) | **Прямой Google Gemini API** (`google_ai_api_key`) |
| 3 | Модель Grok Imagine + ключ | **Прямой xAI API** (`grok_api_key`) |
| 4 | Модель Midjourney + ключ | **Прямой Midjourney API** (`midjourney_api_key`, MidAPI) |
| 5 | Остальные (Flux, Kling, Ideogram и т.д.) | **fal.ai** (`falai_api_key`) |

**Почему часть моделей идёт через fal:** У Flux (Black Forest Labs), Kling (Kuaishou), Ideogram в конфиге нет отдельных ключей — они подключены единым ключом fal. Если появятся прямые ключи (например Replicate для Flux, или официальные API Kling/Ideogram), в `bot/handlers/image.py` в `_route_image_gen` можно добавить проверки по аналогии с xAI и вызывать новые провайдеры из `bot/providers/`.

**Grok Imagine:** при наличии `grok_api_key` — прямой xAI; без ключа — fal.  
**Midjourney:** при наличии `midjourney_api_key` (например [MidAPI](https://midapi.ai)) — реальный Midjourney (асинхронная генерация, опрос статуса); без ключа — fal (Flux Pro как запасной вариант).

---

Все перечисленные модели используются для **генерации по тексту** (кнопка «🎨 Картинка» или текст в чат). Размер задаётся флагами в промпте: `--landscape`, `--square`, `--portrait` (или `--горизонт`, `--квадрат`, `--вертик`). После генерации доступны кнопки **«🔄 Ещё раз»** (регенерация с тем же промптом) и **«🔍 Апскейл»** (fal.ai ESRGAN, +5 CR к любой картинке).

**Редактирование фото** (отправить фото с подписью типа «измени цвет платья»): поддерживается **только модель GPT Image** (OpenAI Edit API). Остальные модели для редактирования недоступны — бот покажет подсказку переключиться на GPT Image.

## 1. Flux Turbo — `flux-2-turbo`
- **Цена:** ⚡ 5 CR (можно тратить бесплатные CR).
- **Провайдер:** fal.ai (FLUX.1 [dev]).
- **Генерация по тексту:** да.
- **Редактирование фото:** нет (при запросе правки — подсказка выбрать GPT Image).
- **Размеры:** альбом (4:3), квадрат (1:1), портрет (3:4).
- **Кол-во картинок за раз:** 1–4.
- **Апскейл:** да (общий апскейл +5 CR).

## 9. GPT Image — `gpt-image`
- **Цена:** 12 CR.
- **Провайдер:** OpenAI (Images Generations + **Images Edit**).
- **Генерация по тексту:** да (quality: high, PNG).
- **Редактирование фото:** **да** — единственная модель, с которой можно отправлять фото и подпись «измени цвет платья» и т.п.; ответ приходит в base64, бот отправляет файл в чат.
- **Размеры:** 1024×1024 (в т.ч. для edit).
- **Кол-во картинок за раз:** до 4 для генерации; для редактирования — 1.
- **Апскейл:** да.
